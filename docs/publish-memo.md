# プライバシー
## 権限が必要な理由
### storage
ユーザーが入力欄のテンプレートやオプションを設定するため。
For users to set input field templates and options.

### ホスト権限(content script)
YouTubeの動画ページ上で動作する拡張機能であるため。
Because it is an extension that runs on the YouTube video page.
