# Thumbnail To Eagle
Browser extension to quickly save YouTube thumbnails to [Eagle](https://eagle.cool/).

## Features

- 🖼️ Display thumbnail on YouTube video pages
- 🦅 Display Save button to Eagle
    - ⚙️ You can setup templates for input
